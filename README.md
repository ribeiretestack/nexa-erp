# Nexa ERP - Dashboard Inteligente 🚀

Um protótipo de ERP (Enterprise Resource Planning) moderno, construído com foco absoluto em **UI/UX design** e na criação de uma interface de alta performance. O sistema simula um ambiente corporativo fluido, dinâmico e responsivo.

🔗 **[Acessar o Projeto ao Vivo](https://ribeiretestack.github.io/nexa-erp/)**

## ✨ Funcionalidades Principais

* **Arquitetura SPA (Single Page Application):** Navegação instantânea entre ecrãs (Início, Vendas, Financeiro, RH, etc.) sem recarregar a página.
* **Gráficos SVG Dinâmicos:** Motor de gráficos Bézier gerado matematicamente com interatividade avançada ao passar o rato (Tooltips responsivos).
* **Módulo de Faturamento em Tempo Real:** Simulação de um carrinho de compras corporativo que injeta e atualiza os dados financeiros do painel instantaneamente.
* **Chat Corporativo com IA:** Interface de comunicação integrada com a "Nexus AI", simulando assistentes virtuais para otimização de rotinas de equipa.
* **Design System & Micro-interações:** Tema escuro nativo (Dark Mode), feedback visual com Toasts, Modais de confirmação, botões expansivos e Skeleton Loaders assíncronos.
* **100% Responsivo:** Layout fluido que se adapta a telemóveis, tablets e monitores (Mobile-First approach).

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido focado na raiz do Front-end, sem dependência de bibliotecas ou frameworks externas pesadas:

* **HTML5:** Semântico e estruturado.
* **CSS3:** Variáveis (Custom Properties), Flexbox, CSS Grid, media queries e animações complexas (Glow effects, transições de estado).
* **JavaScript (Vanilla):** Manipulação avançada do DOM, escuta de eventos, simulação de chamadas assíncronas (API mock) e gestão do estado da aplicação.

## 💡 Sobre o Desenvolvimento

Este MVP (Mínimo Produto Viável) foi desenhado para provar que é possível entregar uma experiência premium, rápida e intuitiva apenas com as tecnologias base da web. Toda a complexidade de backend foi abstraída para priorizar o conforto visual, a velocidade de carregamento e a usabilidade do utilizador final.
